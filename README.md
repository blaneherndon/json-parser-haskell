# json-parser-haskell
This is a simple JSON parser implemented in the Haskell programming language

This project is not finished yet and will not work

### Run the parser with your favorite container program
```sh
# Using podman
$ podman build -t json-parser-haskell .
$ podman run --rm json-parser-haskell

# Using docker
$ docker build -t json-parser-haskell .
$ docker run --rm json-parser-haskell
```
